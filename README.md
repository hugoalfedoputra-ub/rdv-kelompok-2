# rdv-kelompok-2

Proyek ini menggunakan Conda sebagai environment manager. Silakan membuat environment terlebih dahulu dengan file `environment.yml` yang telah disediakan.

Set up proyek ini dengan membuat folder bernama `data` di `root`. Folder tersebut akan berisi data yang diunduh melalui file Notebook atau Python script pada proyek ini. Data ini tidak diperkenankan untuk di-push ke Github.
